## 🚀 使い方

### Chromeにインポートする方法

1. **Chromeを開く**
2. **ブックマークマネージャーを開く**
   - ショートカット: `Ctrl + Shift + O` (Windows/Linux)
   - または: `Cmd + Option + B` (Mac)
3. **右上のメニュー（︙）をクリック**
4. **「ブックマークをインポート」を選択**
5. **`bookmarks_categorized.html`を選択してインポート**

### ⚠️ 注意事項

- **インポート前に現在のブックマークをエクスポートしてバックアップすることを強く推奨します**
- カテゴリー分類は自動なので、必要に応じて手動で調整してください
- 「その他」カテゴリーが32.7%と多めです。必要に応じて手動で再分類してください

## 🛠️ 使用したスクリプト

### `clean_bookmarks.py`
重複・無効URLを削除するスクリプト

```bash
python3 clean_bookmarks.py
```

### `categorize_bookmarks.py`
ブックマークを自動的にカテゴリー分類するスクリプト

```bash
python3 categorize_bookmarks.py
```

### `final_report.py`
最終レポートを生成するスクリプト

```bash
python3 final_report.py
```
